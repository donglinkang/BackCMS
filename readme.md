#BackCMS

基于 Laravel 的CMS,功能非常简洁.

包含功能
* 单页组件
* 文章组件
* 表单组件.
* 多角色权限
* 系统文件组件

##安装方式
* 下载文件
* composer update
* cp .env.example .env && ./artisan key:generate
* 编辑 .env 文件中数据库连接
* 访问 http://xxxxx/ 提示安装
* 安装成功后,会在 /storage/app 生成 installed 文件


##卸载数据
* 访问 /storage/app 下 installed 文件 ,里面有卸载密码.
* 访问 http://xxxxx/install/uninstall/卸载密码
* 卸载成功
